Avatar

The Avatar component is used to represent user, and displays the profile picture, initials or fallback icon. They are found throughout material design with uses in everything from tables to dialog menus.

Import

import { Avatar }from '@celestial-ui/react'

Image Avatar

Image avatars can be created by passing standard img props src or srcSet to the component.

<Avatar alt="Naruto Uzumaki" src=' /static/images/avatar/1.jpg'/>
<Avatar alt="Boa Hancock" src=' /static/images/avatar/2.jpg'/>
<Avatar alt="Ichigo Kurosaki" src=' /static/images/avatar/3.jpg'/>

Letter Avatar

Avatars containing simple characters can be created by passing a string as children.

AS

AH

NH

<Avatar>AS</Avatar>
<Avatar sx={{ bgColor: deepOrange[ 500] }}>AH</Avatar>
<Avatar sx={{ bgColor: deepPurple[ 500] }}>NH</Avatar>

Sizes

You can change the size of the avatar with the height and width CSS properties.

<Avatar alt="Naruto Uzumaki" src=' /static/images/avatar/1.jpg'
sx={{ width: 24,height: 24 }}/>
<Avatar alt="Boa Hancock" src=' /static/images/avatar/2.jpg'
sx={{ width: 56,height: 56 }}/>
<Avatar alt="Ichigo Kurosaki" src=' /static/images/avatar/3.jpg'/>
