Header

Heading is used to render semantic HTML heading elements.

Import

import { Heading }from '@celestial-ui/react'

Usage

Heading composes Box so you can use all the style props and add responsive styles as well. It renders an <h2> tag by default.

(4xl)I love Celestial Dragon & React

(3xl)I love Celestial Dragon & React

(2xl)I love Celestial Dragon & React

(xl)I love Celestial Dragon & React

(lg)I love Celestial Dragon & React

(md)I love Celestial Dragon & React

(sm)I love Celestial Dragon & React

<Heading as='h1' fontSize="4xl">(4xl)I love Celestial Dragon & React</Heading>
<Heading as='h2' fontSize="3xl">(3xl)I love Celestial Dragon & React</Heading>
<Heading as='h2' fontSize="2xl">(2xl)I love Celestial Dragon & React</Heading>
<Heading as='h2' fontSize="xl">(xl)I love Celestial Dragon & React</Heading>
<Heading as='h3' fontSize="lg">(lg)I love Celestial Dragon & React</Heading>
<Heading as='h4' fontSize="md">(md)I love Celestial Dragon & React</Heading>
<Heading as='h5' fontSize="sm">(sm)I love Celestial Dragon & React</Heading>

Override Style

You can change the entire style of the text via props. For example, to change the font size, pass the fontSize prop. No need to write css or styled().

I'm using a custom font-size value...

<Heading size='lg' fontSize="50px" color='tomato'>I'm using a custom font-size value for this text</Text>
